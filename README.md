## 特性
- 使用 TypeScript 构建，提高开发质量
- SPA页面应用，让页面架构简洁但不简单
- webpack打包，让代码安全的面对用户，是代码更轻

## 支持环境
* 现代浏览器和 IE9 及以上。
* 支持服务端渲染。

## 本地环境支持
* @angular/cli: 1.7.3
* typescript: 2.5.3
* webpack: 3.11.0
* Node: 9.7.1
* Angular: 5.2.9

## 需要技能
* es6
* angular2+
* typescript
* webpack
* sass

## 快速启动
### 安装angular-cli
```bash
npm install -g @angular/cli
```
### 克隆项目到本地
```bash
git clone https://github.com/DocStackTop/DocStack-web.git
```
### 进入本地目录
```bash
cd DocStack-web
```
### 安装依赖
```bash
npm install
```
### 启动项目,在浏览器中输入localhost:4200运行
```bash
ng server
```
### 打包
```bash
ng build --prod --aot --base-href ./
```
